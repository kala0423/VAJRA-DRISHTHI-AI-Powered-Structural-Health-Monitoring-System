#  Project Overview

VAJRA-DRISHTHI is an Artificial Intelligence and Computer Vision based structural monitoring system developed as a student research project.

The system analyzes images of buildings and infrastructure to detect cracks, estimate their dimensions, and generate inspection reports.

This project is intended for:

• Academic research
• Student learning
• Prototype structural monitoring
• Computer vision experimentation

---

#  Features

• Automated crack detection using OpenCV
• Crack width and depth estimation (simulated)
• Risk level classification
• Image-based structural analysis
• Report generation (Text / PDF)
• Crack highlighting visualization

# ⚙️ Installation

Clone repository:

```
git clone https://github.com/YOUR_USERNAME/vajra-drishthi.git
cd vajra-drishthi
```

Install dependencies:

```
pip install -r requirements.txt
```

Run application:

```
python vajra_complete.py
```

#  Usage

1. Open the application
2. Upload structural image
3. System analyzes cracks
4. View crack measurements
5. Generate inspection report
